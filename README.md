# P[y]TAB

Repository dell'edizione mattutina del corso di Python per il PTAB.

Nella directory [lezioni](lezioni) trovate i *notebook* con il materiale delle lezioni, mentre nella directory [sito](sito) i curiosi possono trovare il codice sorgente del [sito web del corso](https://mapio.github.io/pytab-2026/), che è stato realizzato usando [MkDocs](https://www.mkdocs.org/).
